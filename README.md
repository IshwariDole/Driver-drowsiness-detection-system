🚗 Driver Drowsiness Detection System

AI-based real-time driver monitoring system that detects drowsiness and sends emergency alerts.

🔍 Features
Real-time face & eye tracking using OpenCV + dlib
Detects:
Sleeping 😴
Drowsiness 😵‍💫
Yawning 🥱
Head tilt 🤕
Looking away 👀
Live video streaming using Flask
Driver registration & MongoDB database
Real-time socket alerts
🚨 Automatic WhatsApp emergency message to emergency contact
🧠 Tech Stack
Python
OpenCV
dlib
Flask
MongoDB
SocketIO
Computer Vision & Machine Learning
